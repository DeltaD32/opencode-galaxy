# Action Labels

> **Source**: BMW Group Density Design System
> For the most up-to-date guidelines, visit:
> https://density.bmwgroup.net/v11/content/voice-and-tone

---

This list includes the common UI terms and recommended action labels for use in BMW Group business applications.

| English      | German           | Description                                                                                                                                                                                                   |
| ------------ | ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Add          | Hinzufügen       | Takes an existing object and uses it in a new context (for example, Add user)                                                                                                                                 |
| Apply        | Anwenden         | Saves changes without closing the dialog (for example, Apply filter)                                                                                                                                          |
| Approve      | Bestätigen       | Indicates the user agrees. In a business process, typically initiates the next step                                                                                                                           |
| Back         | Zurück           | Returns the user to the previous step in a sequence of steps, such as in a wizard                                                                                                                             |
| Browse       | Durchsuchen      | Assists the user in selecting a file (for example, on a button or link next to an entry field)                                                                                                                |
| Cancel       | Abbrechen        | Stops the current action and closes the dialog                                                                                                                                                                |
| Clear        | Zurücksetzen     | This action clears all the fields or selections (for example, Clear all or Clear fields, Clear Filter)                                                                                                        |
| Close        | Schließen        | Closes the current page or window                                                                                                                                                                             |
| Copy         | Kopieren         | Creates new instances of the selected objects (for example, Copy folder)                                                                                                                                      |
| Create       | Erstellen        | Makes a new object from scratch (for example, create new document)                                                                                                                                            |
| Customize    | Anpassen         | Allow a user to make desired changes                                                                                                                                                                          |
| Delete       | Löschen          | Destroys an existing object so that it no longer exists (for example, Delete column)                                                                                                                          |
| Done         | Fertig           | Indicates that the user has finished working in an environment (for example, editing templates) and wants to return to where he or she came from                                                              |
| Download     | Herunterladen    | Transfers a file from a remote system to a local system                                                                                                                                                       |
| Edit         | Bearbeiten       | Allows data or values to be changed (for example Edit document)                                                                                                                                               |
| Export       | Exportieren      | Saves data in a different format external to the system (for example, export data)                                                                                                                            |
| Filter       | Filtern          | Shortens a list to objects that match the filter criteria                                                                                                                                                     |
| Finish       | Abschließen      | Indicates completion of a series of steps, such as in a wizard                                                                                                                                                |
| Get help     | Hilfe            | Opens page, where the user can search for help information                                                                                                                                                    |
| Hide         | Ausblenden       | Removes an element that was previously shown (for example, hide details)                                                                                                                                      |
| Import       | Importieren      | Transforms data or objects from an external source (for example, import data)                                                                                                                                 |
| Insert       | Einfügen         | Adds an element at a particular position in an ordered view (for example, insert column)                                                                                                                      |
| Learn more   | Mehr erfahren    | Opens additional, highly contextual information. Insert at the end of inline text or over text where more information follows but does not fit in the current context (for example, Learn more about regions) |
| Log in       | Anmelden         | Enters a site or application                                                                                                                                                                                  |
| Log out      | Abmelden         | Exits an application or site                                                                                                                                                                                  |
| Move         | Verschieben      | Transfers an object from one container to another (for example, Move to trash)                                                                                                                                |
| New          | Neu              | Starts the creation of a new object (for example, New user or New column)                                                                                                                                     |
| Next         | Weiter           | Advances the user to the next step in a sequence of steps, such as in a wizard                                                                                                                                |
| OK           | OK               | Confirms an action or completes the current task. Best practice is to use a label corresponding to the specific action (for example, Save or Close or Delete). Use OK only when such a label is not available |
| Play         | Abspielen        | Starts audio, video, or an animation. It is also possible to use 'Play' in German                                                                                                                             |
| Post         | Teilen           | Adds a new comment to an online community or adds status to a log or record                                                                                                                                   |
| Preview      | Vorschau         | Shows how an object or content will appear with formatting applied before the content is published or distributed                                                                                             |
| Print        | Drucken          | Sends a copy of the currently selected object or the object in view to the printer                                                                                                                            |
| Redo         | Wiederherstellen | Redoes an undo action                                                                                                                                                                                         |
| Refresh      | Aktualisieren    | Reloads the view of an object when the displayed view has become unsynchronized with the source                                                                                                               |
| Reject       | Ablehnen         | Indicates the user does not approve. In a business process, typically blocks the process from proceeding to the next step                                                                                     |
| Remove       | Entfernen        | Removes an object from the current context but the object is not destroyed as a result of the action (for example, removes a user from a group)                                                               |
| Reply        | Antworten        | Indicates or completes a response to an email or a comment                                                                                                                                                    |
| Reset        | Zurücksetzen     | Reverts values back to their last saved state                                                                                                                                                                 |
| Restore      | Wiederherstellen | Brings a file back after deletion, corruption, or similar event. Do not use restore in combination with undo                                                                                                  |
| Run          | Ausführen        | Initiates a procedure. Use Run instead of Execute                                                                                                                                                             |
| Save         | Speichern        | Saves pending modifications made to a file or document. Does not close the window or panel                                                                                                                    |
| Save as      | Speichern als    | Creates a new object based on the state of the object currently being viewed. The user names the new object and typically identifies its location                                                             |
| Search       | Suchen           | Returns all objects (for example, files, names, or documents) within a defined set (for example, in a folder, directory, database, or the internet) that match some specified criteria                        |
| Select       | Auswählen        | Selects data from a table                                                                                                                                                                                     |
| Select all   | Alles auswählen  | Adds all objects in the view to the selection set or checks all checkboxes                                                                                                                                    |
| Send         | Absenden         | Transfers an email or other information to the recipient or destination                                                                                                                                       |
| Show         | Anzeigen         | Reveals an object that was previously hidden (for example, shows descriptions or shows further details)                                                                                                       |
| Show less    | Weniger anzeigen | Removes additional information that was previously shown. Usually used at the end of a description. (for example, hides some content of a description)                                                        |
| Show more    | Mehr anzeigen    | Reveals additional information that was previously hidden. Usually used at the end of a description for saving space. (for example, shows the full content of a description)                                  |
| Sign up      | Registrieren     | Creates a user account or registers a user in a system                                                                                                                                                        |
| Sort         | Sortieren        | Sorts a list or table column                                                                                                                                                                                  |
| Top          | Nach oben        | Returns to the top of the page                                                                                                                                                                                |
| Undo         | Rückgängig       | Reverts to the state before the most recent changes made by the user                                                                                                                                          |
| Update       | Aktualisieren    | Label for a button in a dialog or form for editing an object. The settings in the dialog are applied to the object when it is updated                                                                         |
| Upload       | Hochladen        | Transfers a file from a local system to a remote system                                                                                                                                                       |
| View details | Details anzeigen | Presents additional information or properties for the object                                                                                                                                                  |
