# Business & Product Icons Reference

All icons use the pattern `al-ds-icon-{name}`. Icons with `_24` suffix are 24px variants.

## Usage

```html
<span class="al-ds-icon-ALP_0001_Operating_Lease"></span>
```

## Alphabet Application Icons (ALA_)

|---|---|
| `ALA_0001_Contract_Revision_App_24` | Contract Revision |
